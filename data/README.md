# Data 1

-   **all_recipes**: Data about recipes on the site containing nutrition information, cook time, and reviews. Contains 14,426 rows, each representing a single recipe.

# Codebook for all_recipes Dataset

## Variable Names and Descriptions:

-   **name**: Name of the recipe
-   **url**: Web address of the recipe page
-   **author**: Username of the recipe's author
-   **date_published**: Date the recipe was published
-   **ingredients**: List of ingredients used in the recipe
-   **calories**: Calories per serving
-   **fat**: Fat content per serving (grams)
-   **carbs**: Carbohydrate content per serving (grams)
-   **protein**: Protein content per serving (grams)
-   **avg_rating**: Average user rating (out of 5)
-   **total_ratings**: Total number of ratings submitted
-   **reviews**: Number of written reviews submitted
-   **prep_time**: Preparation time (minutes)
-   **cook_time**: Cooking time (minutes)
-   **total_time**: Total time to make the recipe (minutes)
-   **servings**: Number of servings the recipe makes

## Data Types:

-   **name**: character
-   **url**: character
-   **author**: character
-   **date_published**: date
-   **ingredients**: character
-   **calories**: integer
-   **fat**: integer
-   **carbs**: integer
-   **protein**: integer
-   **avg_rating**: double
-   **total_ratings**: integer
-   **reviews**: integer
-   **prep_time**: integer
-   **cook_time**: integer
-   **total_time**: integer
-   **servings**: integer
